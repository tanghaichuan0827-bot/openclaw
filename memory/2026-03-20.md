## 微信发送消息经验

### 关键发现
- **微信 Mac 发消息用回车键**，不是 Cmd+Enter
- 微信有防录屏保护，screencapture 会显示绿色遮罩
- peekaboo 的 `see` 命令可以绕过防录屏获取界面截图

### peekaboo 命令语法
```bash
# 聚焦窗口
peekaboo window focus --app "微信"

# 调整窗口大小
peekaboo window set-bounds --app "微信" --x 0 --y 0 --width 1200 --height 900

# 输入文字（注意：文字直接写，不用 --text 参数）
peekaboo type "文字" --app "微信"

# 快捷键
peekaboo hotkey --keys "cmd,f" --app "微信"

# 点击坐标
peekaboo click --app "微信" --coords "x,y"

# 查看UI元素
peekaboo see --app "微信"
```

### 操作流程（给糖糖发消息）
1. 聚焦微信 → 调整窗口
2. Cmd+F 搜索联系人
3. 输入名字 → 回车选择
4. 输入消息 → 回车发送

### AppleScript 方式（更稳定）
```applescript
tell application "System Events"
    tell process "WeChat"
        set frontmost to true
        keystroke "f" using {command down}
        delay 0.5
        keystroke "糖糖"
        delay 1
        keystroke return
        delay 0.5
        keystroke "在吗"
        delay 0.3
        keystroke return
    end tell
end tell
```
注意：AppleScript 中写中文可能有编码问题，建议用临时文件保存脚本再执行。

## 微信发送消息 - 完美流程（已验证）

### 成功方案：peekaboo + AppleScript 混合
```bash
# 1. 聚焦微信
peekaboo window focus --app "微信"
sleep 0.3

# 2. ESC清空当前状态
peekaboo hotkey --keys "escape" --app "微信"
sleep 0.3

# 3. Cmd+F 打开搜索
peekaboo hotkey --keys "cmd,f" --app "微信"
sleep 0.5

# 4. 清空搜索框
peekaboo hotkey --keys "cmd,a" --app "微信"
sleep 0.1
peekaboo hotkey --keys "delete" --app "微信"
sleep 0.2

# 5. 输入联系人名字（用 peekaboo type 避免中文编码问题）
peekaboo type "联系人名字" --app "微信"
sleep 1.5

# 6. 回车选中联系人
peekaboo hotkey --keys "return" --app "微信"
sleep 1

# 7. 输入消息
peekaboo type "消息内容" --app "微信"
sleep 0.5

# 8. 回车发送（微信Mac用回车发送，不是Cmd+Enter）
peekaboo hotkey --keys "return" --app "微信"
```

### 关键要点
- **中文输入用 `peekaboo type`**，AppleScript 直接写中文会编码错误
- **搜索后等待1.5秒**再回车，确保搜索结果加载
- **选中联系人后等待1秒**再输入，确保聊天窗口打开
- **发送用回车键**，不是 Cmd+Enter
- **每次操作前先 ESC + 搜索**，避免之前的状态干扰
